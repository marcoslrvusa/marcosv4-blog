## 3. SEO + AEO/GEO — Share of AI Voice

### 3.1 Conceito: Share of AI Voice

**Definição:** Frequência com que o nome, marca ou conteúdo de Marcos Luciano aparece como fonte/citação nas respostas de IAs (ChatGPT, Perplexity, Claude, Gemini, Copilot).

Métrica composta:
- **Citações diretas:** "Segundo Marcos Luciano..." / "De acordo com marcosv4.cloud..."
- **Citações indiretas:** Conteúdo do blog parafraseado sem link (mais difícil de medir)
- **Links nos responses:** Frequência com que links do blog aparecem em respostas AI

### 3.2 Técnicas Avançadas de AEO/GEO

#### 3.2.1 Estrutura de Artigo Otimizada para IAs

```
Título: [Palavra-chave principal]: [Benefício/Diferenciação] + [Ano]
Meta: <135 chars com call-to-action informacional
Schema: Article + FAQPage + HowTo (quando aplicável)
Headings: Hierarquia lógica com perguntas naturais
Primeiro parágrafo: Resposta direta à pergunta principal (<60 palavras)
Listas e tabelas: Dados estruturados que IAs adoram extrair
FAQ no final: 3-5 perguntas relacionadas em schema FAQPage
Citações e fontes: Links para fontes confiáveis (aumenta credibilidade aos olhos das IAs)
```

#### 3.2.2 Perguntas-Chave para Otimizar

Para cada tópico, identificar as **10 perguntas que usuários fazem** (AnswerThePublic, AlsoAsked, People Also Ask) e responder diretamente no artigo.

Exemplo para tópico "AI Search":
- O que é AI Search Optimization?
- Como funciona o GEO (Generative Engine Optimization)?
- Qual a diferença entre SEO tradicional e AEO?
- Como otimizar conteúdo para ChatGPT Search?
- Empresas brasileiras devem investir em AEO em 2026?

**Mapa de Topical Authority — Marcos Luciano:**

### 3.3 Backlinks Estratégicos para Confiança AI

IAs confiam mais em domínios que também são confiáveis para Google. Priorizar backlinks de:

| Tipo de Domínio | Exemplos | Peso AI |
|----------------|----------|---------|
| Educacional (.edu) | USP, Unicamp, FGV, MIT | Altíssimo |
| Governamental (.gov) | Governo do Brasil, Câmara dos Deputados | Altíssimo |
| Mídia tradicional | Estadão, Folha, Globo, Exame, InfoMoney | Alto |
| Tech publishers | Hashnode, Dev.to, Medium, Towards Data Science | Alto |
| Institutos tech | DataCamp, Google, Microsoft, AWS | Altíssimo |
| Podcasts e entrevistas | Flow, PodPah, Inteligência Ltda., Tecnologia Aberta | Alto |

**Ações concretas:**
1. **Guest posts** em veículos como Hashnode, DataCamp Blog, Medium (publicar versão adaptada)
2. **Entrevistas em podcasts** → pedir link no show notes
3. **Ser citado como fonte** → oferecer dados exclusivos para jornalistas (Exame, Startups, Neofeed)
4. **Citações acadêmicas** → oferecer mentoria gratuita a alunos de pós-graduação em IA em troca de citação
5. **Parcerias com .edu** → palestras em universidades (USP, FGV, Mackenzie, PUC)

### 3.4 Técnicas de AEO Avançadas

**Entity SEO / Knowledge Graph:**
- Garantir que o schema `Person` do blog esteja perfeitamente preenchido
- Criar página de autor separada em `/about` com biografia completa, linkedIn, GitHub
- Submeter a Wikipedia-like sources (Crunchbase, Wikidata)

**Multimodal Indexing:**
- Todo artigo com imagem deve ter alt text descritivo
- Incluir diagramas e infográficos em formato que IAs possam interpretar (SVG com texto acessível)
- Transcrições completas em vídeos

**Citation Optimization:**
- Citar fontes autoritativas em cada artigo (datas, estatísticas com links)
- IAs dão preferência para conteúdo que cita fontes primárias (papers, dados oficiais)

## 8. AdSense Estratégico no Blog

### 8.1 Filosofia: Receita Passiva SEM Poluir UX

O blog é o ativo principal de autoridade. Anúncios NÃO podem:
- Interferir na leitura
- Competir com CTAs de consultoria
- Quebrar a experiência premium/mínima do dark theme V4
- Ser intrusivos (pop-ups, banners fullscreen)

### 8.2 Locais Aprovados

| Local | Formato | Descrição | Impacto UX |
|-------|---------|-----------|------------|
| **Sidebar (após tópicos)** | Display retangular (300x250) | Abaixo da seção de tópicos, antes da newsletter | **Baixo** — sidebar já existe |
| **Entre parágrafos (após 3º parágrafo)** | In-article (300x250 ou responsivo) | Após o 3º parágrafo de artigos longos (>500 palavras) | **Médio** — aceitável se fluído |
| **Fim do artigo (antes do FAQ)** | Display horizontal (728x90) | Entre o fim do artigo e a seção de FAQ | **Muito Baixo** — quase imperceptível |
| **Página inicial / listagem** | In-feed (entre posts 3 e 4) | Entre o 3º e 4º post da listagem | **Baixo** — nativo no feed |
| **Rodapé** | Display horizontal (728x90) | Antes do footer | **Muito Baixo** — não atrapalha |
| **Página de tópico** | In-feed | Entre resultados | **Baixo** — similar a listagem |

### 8.3 Formatos de Anúncio Recomendados

| Formato | Onde usar | Vantagem |
|---------|-----------|----------|
| **Display responsivo** | Sidebar, in-article | Adapta ao layout, não quebra design |
| **In-article** | Entre parágrafos | Anúncio nativo no conteúdo |
| **In-feed** | Listagem de posts | Parece parte do feed |
| **Anúncio nativo** | Sidebar | Menos intrusivo |
| **Matched content** | Fim do artigo | Recomenda outros posts + anúncios |

### 8.4 Formatos a EVITAR

- **Anchor/overlay ads** (fixos no topo/fundo) — poluem, afastam lead qualificado
- **Pop-ups** — experiência terrível
- **Vídeos com som** — quebram imersão
- **Expansíveis** — irritantes

### 8.5 Estimativa de Receita AdSense

Parâmetros estimados:
- Tráfego atual: ~1.000-3.000 visitas/mês (140+ artigos)
- RPM Brasil tech: ~R$8-15
- RPM países EN: ~$5-15 USD

| Cenário | Tráfego | RPM | Receita Mensal |
|---------|---------|-----|----------------|
| Atual | 2k visits/mês | R$10 | R$20-30 |
| Médio prazo (6 meses) | 10k visits/mês | R$12 | R$120-150 |
| 1 ano | 50k visits/mês | R$15 | R$750-900 |
| 2 anos | 200k visits/mês | R$15 | R$3.000-4.500 |

**Nota:** AdSense é complementar. O foco financeiro é consultoria. Anúncios pagam o hosting e café.

### 8.6 Implementação Técnica AdSense

1. Criar conta AdSense (se já não tiver)
2. Adicionar site ao AdSense (verificar propriedade)
3. Gerar código de anúncio
4. Criar componente React `AdUnit.tsx` no blog:

```tsx
// src/components/AdUnit.tsx
"use client"
import { useEffect, useRef } from "react"

interface AdUnitProps {
  slot: string
  format?: "rectangle" | "horizontal" | "vertical" | "responsive"
  className?: string
}

export default function AdUnit({ slot, format = "responsive", className = "" }: AdUnitProps) {
  const adRef = useRef<HTMLDivElement>(null)

  useEffect(() => {
    if (typeof window !== "undefined" && adRef.current) {
      try {
        ((window as any).adsbygoogle = (window as any).adsbygoogle || []).push({})
      } catch (e) {
        console.error("AdSense error:", e)
      }
    }
  }, [])

  const sizes = {
    rectangle: { width: 300, height: 250 },
    horizontal: { width: 728, height: 90 },
    vertical: { width: 160, height: 600 },
    responsive: { width: "100%", height: "auto" },
  }

  const size = sizes[format]

  return (
    <div className={className}>
      <ins
        className="adsbygoogle"
        style={{ display: "block", ...size }}
        data-ad-client="ca-pub-SEU_ID"
        data-ad-slot={slot}
        data-ad-format={format === "responsive" ? "auto" : format}
        data-full-width-responsive="true"
      />
    </div>
  )
}
```

5. Inserir nos locais definidos (Sidebar, PostPage, listagem)
6. Configurar block controls para não exibir anúncios em:
   - Página de Consultoria
   - Página de Obrigado (pós-lead magnet)
   - Páginas de e-book

## Notas Finais

1. **Consistência > Perfeição.** 80% pronto e publicado é melhor que 100% pronto e nunca publicado.
2. **Dados próprios são seu maior ativo.** Cases, métricas, benchmarks que VOCÊ gerou são o que nenhuma IA consegue replicar.
3. **Posicionamento de preço é posicionamento de marca.** Quanto mais caro, mais autoridade. Não tenha medo de cobrar.
4. **O blog é o centro do ecossistema.** Todo conteúdo de todas as plataformas deve levar de volta ao blog (link na bio, CTA nos posts).
5. **AI Search é a fronteira.** Estar bem posicionado em ChatGPT, Perplexity e Google AI Overviews é o diferencial competitivo para 2026-2027.
6. **Personal branding não é vaidade — é ativo de negócio.** Cada seguidor qualificado é um lead em potencial. Trate a audiência com respeito e entregue valor real.
